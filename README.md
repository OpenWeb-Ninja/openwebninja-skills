# OpenWeb Ninja Skills

A collection of Claude Code skills for the [OpenWeb Ninja](https://openwebninja.com) API platform.

## Skills

| Skill | Description |
|-------|-------------|
| [openwebninja-universal-scraper](./openwebninja-universal-scraper) | Extract data from 35+ public web APIs using natural language — lead generation, market research, competitor analysis, finance, real estate, and more |

## What are Claude Code Skills?

Skills extend Claude's capabilities with reusable, task-specific instructions and scripts. Drop a skill into your `.claude/skills/` folder and Claude automatically knows how to use it.

Learn more: [Claude Code Skills Docs](https://code.claude.ai/docs/en/skills)
